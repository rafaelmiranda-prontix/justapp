# Roteiro de comunicação – Advogados selecionados para o Beta

Este documento contém o texto sugerido para enviar aos advogados **após** a aprovação no painel admin (ex.: via WhatsApp ou e-mail, usando N8N ou outro canal).

## Canal

- **Principal:** WhatsApp  
- **Opcional:** E-mail (mesmo conteúdo)

## Conteúdo da mensagem

1. Saudação com o nome do advogado.
2. **"Você foi selecionado para o teste beta."**
3. **"Desculpe a demora – estamos com alta demanda de pedidos e cadastros."**
4. **"Mas você foi selecionado para participar do teste beta."**
5. Incluir o **link da regulamentação do teste beta:**  
   `https://[SEU-DOMINIO]/em-teste`
6. (Opcional) Pedir confirmação: *"Deseja realmente participar do programa Beta?"* com instrução para responder Sim/Não no WhatsApp.

## Assunto (se e-mail)

**Exemplo:** JustApp Beta – Você foi selecionado para participar

## Exemplo completo (WhatsApp)

```
Olá, [Nome do Advogado]!

Você foi selecionado para o teste beta.

Desculpe a demora – estamos com alta demanda de pedidos e cadastros. Mas você foi selecionado para participar do teste beta.

Regulamentação do teste beta: https://[SEU-DOMINIO]/em-teste

Deseja realmente participar do programa Beta? Responda Sim ou Não.
```

## Próximos passos (N8N / operação)

- Conforme a resposta (se houver confirmação), marcar no CRM/planilha ou disparar próximo passo (ex.: link de acesso, código de convite).
- A **aprovação** do advogado segue sendo feita no painel admin; este roteiro é apenas a comunicação pós-aprovação.
